# Contributing to Moyasar & Ecwid Integration

First off, thank you for considering contributing to this project! It's people like you that make the open-source community such an amazing place.

## Code of Conduct

By participating in this project, you are expected to uphold our Code of Conduct (be kind and professional).

## How Can I Contribute?

### Reporting Bugs
- Use the GitHub Issues tracker.
- Describe the bug in detail and provide steps to reproduce.
- Include information about your environment (Node version, OS).

### Suggesting Enhancements
- Open a GitHub Issue with the tag "enhancement".
- Explain the benefit of the feature and how it should work.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. Follow the existing code style (clean architecture, use controllers/services).
3. Update the documentation if you're making API changes.
4. Ensure your code passes all lint checks (if applicable).

## Development Flow
- Use `npm run dev` for local testing.
- Keep your changes small and focused in a single PR.
- Use meaningful commit messages.

## Questions?
Feel free to open an issue for any questions regarding the codebase.
